## 2. 为什么不用 OpenAI Codex

### ❌ OpenAI API 在国内的使用障碍

#### 网络访问限制
- OpenAI API 在大陆**无法直接访问**
- 必须使用**科学上网工具** (VPN/代理)
- VPN 稳定性**影响服务可用性**
- 企业环境可能**禁止使用 VPN**

#### 支付方式限制
- **不支持** 大陆银行卡支付
- 需要 **Visa/MasterCard** 国际信用卡
- 或使用**虚拟信用卡** (额外成本)
- 部分地区可能面临**封号风险**

### 🔧 核心技术组件

#### 1. **OpenClaw** - AI 助手框架

**版本**: 2026.4.9  
**作用**: 统一的 AI 助手开发框架

**核心功能**:
- ✅ **Gateway 服务** - HTTP API 网关，端口 18789
- ✅ **Agent 编排** - 管理对话流程和工具调用
- ✅ **Skills 系统** - 可插拔的技能模块
- ✅ **多端支持** - Web/CLI/微信等多种交互方式
- ✅ **安全沙箱** - 工具调用权限控制

**配置文件**: `~/.openclaw/openclaw.json`

#### 2. **Ollama** - 本地 LLM 运行环境

**版本**: 最新版  
**作用**: 在本地运行大语言模型

**核心特性**:
- ✅ **模型管理** - 下载、切换、删除模型
- ✅ **Metal 加速** - 充分利用 Apple Silicon GPU
- ✅ **HTTP API** - 端口 11434，兼容 OpenAI 格式
- ✅ **流式输出** - 支持实时响应
- ✅ **上下文管理** - 高效的内存管理

#### 3. **Qwen3.5** - 阿里多模态大模型

**版本**: 7B 参数版本  
**作用**: 提供智能对话和工具调用能力

**模型特性**:
- 🎯 **多模态能力** - 文本、图像理解
- 🚀 **中文优化** - 业界领先的中文理解
- 💡 **推理强大** - 复杂逻辑、数学计算、代码生成
- 🔧 **Function Calling** - 原生支持工具调用
- ⚡ **开源免费** - 完全本地部署
- 🍎 **Apple Silicon 优化** - M 系列芯片优化

#### 4. **taobao-native** - 淘宝 MCP 服务

**版本**: 官方最新版  
**作用**: 提供淘宝购物原生接口

**提供的工具**:
- 🔍 `taobao_search` - 商品搜索
- 📄 `taobao_detail` - 商品详情
- 🛒 `taobao_add_cart` - 加入购物车
- 💳 `taobao_checkout` - 结算订单
- 📦 `taobao_orders` - 订单查询
- 👀 `taobao_watch_price` - 价格监控

#### 5. **微信 ClawBot** - 微信机器人插件

**版本**: 官方最新版
**作用**: 通过微信与 AI 助手对话

**核心功能**:
- 📱 **原生集成** - 微信内直接对话
- 🔐 **端到端加密** - 本地处理，安全可靠
- 🚀 **实时响应** - 快速回复消息
- 🛍️ **完整功能** - 支持所有购物流程
- 🔔 **推送通知** - 价格监控提醒

### 📊 数据流向

```
用户输入 "帮我找个机械键盘"
    ↓
前端 (Web/CLI/微信)
    ↓ HTTP Request
OpenClaw Gateway
    ↓ 解析意图
Ollama (Qwen3.5)
    ↓ 决策调用 taobao_search
taobao-native MCP
    ↓ 访问淘宝数据
淘宝桌面客户端
    ↓ 返回搜索结果
taobao-native → Ollama → Gateway → 前端
    ↓
展示商品列表给用户
```

## 5. 部署流程（简化）

### 阶段一：核心组件
1. **安装 Ollama** - 本地大模型运行环境
2. **下载 Qwen3.5** - 9B 参数多模态模型（约 10GB）
3. **安装 OpenClaw** - AI 助手框架

### 阶段二：淘宝集成
1. **安装淘宝桌面客户端** - 2.5.1+ 版本
2. **启用 MCP 服务** - 在淘宝设置中开启
3. **配置 Skills** - 下载并安装 taobao-native Skill

### 阶段三：多端访问（可选）
1. **Web UI** - 浏览器访问本地服务
2. **微信 ClawBot** - 一键安装微信机器人插件

### 💡 使用技巧总结

#### ✅ 推荐使用方式

1. **明确表达需求** - "300-500 元的机械键盘"
2. **指定筛选条件** - "要销量高的、天猫店"
3. **参考编号** - "看第 2 个"、"买第一个"
4. **确认敏感操作** - AI 会主动要求确认支付等操作

#### ⚠️ 注意事项

1. **支付需手动完成** - AI 仅协助下单，不会自动支付
2. **账号需登录** - 确保淘宝客户端已登录
3. **网络要稳定** - 本地环境，不依赖外网
4. **数据完全私密** - 所有对话本地处理
